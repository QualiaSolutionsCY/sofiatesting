# Listing Reviewer Assignments

## Overview

The listing reviewer fields determine WHO sees the draft property in their "My Draft Properties" dashboard. There can be up to 2 reviewers.

---

## FOR SALE Properties

### Paphos, Limassol, Larnaca, Nicosia

| Field | Value |
|-------|-------|
| Listing Reviewer 1 | `listings@zyprus.com` (Lauren - ALWAYS) |
| Listing Reviewer 2 | Regional manager account (see below) |
| Listing Owner | Agent who sent the instructions |
| Listing Instructor | Agent who sent the instructions |

**Regional Manager Accounts for Reviewer 2:**
- Paphos: `requestpaphos@zyprus.com`
- Limassol: `requestlimassol@zyprus.com`
- Larnaca: `requestlarnaca@zyprus.com`
- Nicosia: `requestnicosia@zyprus.com`

**Rationale:** Lauren is always Reviewer 1 for sales. Reviewer 2 is the regional account so that if Lauren is sick or away, the regional manager can also review and publish.

### Famagusta ONLY

| Field | Value |
|-------|-------|
| Listing Reviewer 1 | `requestfamagusta@zyprus.com` (regional manager) |
| Listing Reviewer 2 | **NONE** (leave empty) |
| Listing Owner | Agent who sent the instructions |
| Listing Instructor | Agent who sent the instructions |

**Rationale:** Famagusta does NOT include Lauren. Only the regional manager reviews.

## FOR RENT Properties

### All Regions (Universal Rule)

| Field | Value |
|-------|-------|
| Listing Reviewer 1 | **Same as agent who sent it** |
| Listing Reviewer 2 | **NONE** (leave empty) |
| Listing Owner | Agent who sent the instructions |
| Listing Instructor | Agent who sent the instructions |

**Rationale:** For rentals, the agent who uploads is also the one who reviews their own listing. No management oversight needed.

## Multiple Reviewers Format

When entering multiple reviewers in the system, use **comma separation**:
```
listings@zyprus.com, requestpaphos@zyprus.com
```

The system has auto-complete functionality for email selection.

---

## What Reviewers Can Do

Once a property appears in their "My Draft Properties" dashboard, reviewers can:

1. **Publish Draft** - Make the listing live on the website
2. **Save Draft** - Keep for later review/editing
3. **Delete Draft** - Remove the listing entirely
