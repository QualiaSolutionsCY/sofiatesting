## ALWAYS DO ✅

- Upload as DRAFT (never publish directly)
- Check for duplicates before every upload
- Tick "AI Generated" checkbox
- Put owner details in My Notes ONLY
- Place map pin 2-3 streets away (privacy)
- Set Negotiable to YES by default
- Generate description from template
- Order images: External → Pool → Interior → Bathrooms

---

## NEVER DO ❌

- Upload outside agent's region (unless Charalambos/Lauren FOR SALE)
- Let Charalambos/Lauren upload rentals
- Put owner name/phone in public description
- Place map pin at exact address
- Copy-paste description from chat
- Skip duplicate check
- Publish directly (always draft)

---

## MINIMUM REQUIRED INFO

1. Price
2. Property type
3. Location/Area
4. Bedrooms & Bathrooms
5. Covered area (sqm)
6. Owner name & phone
7. Title deed status
8. At least 1 photo
