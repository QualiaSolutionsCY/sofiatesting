  <action>
Replace the current Promise.all loop (lines 63-81) with a single JOIN query:

**Current pattern (N+1):**
```typescript
// Gets all chats (1 query)
const userChats = await db.select(...).from(chat)...

// Gets messages for each chat (N queries)
const chatsWithMessages = await Promise.all(
  userChats.map(async (chatItem) => {
    const messages = await db.select(...).from(message)...
    return { ...chatItem, messages };
  })
);
```

**Optimized pattern (JOIN):**
```typescript
import { desc, eq, sql } from "drizzle-orm"; // Add sql to imports

// Single query with LEFT JOIN + GROUP BY
const chatsWithMessages = await db
  .select({
    id: chat.id,
    title: chat.title,
    createdAt: chat.createdAt,
    visibility: chat.visibility,
    messages: sql<Array<{
      id: string;
      role: string;
      parts: unknown;
      createdAt: Date;
    }>>`
      COALESCE(
        json_agg(
          json_build_object(
            'id', ${message.id},
            'role', ${message.role},
            'parts', ${message.parts},
            'createdAt', ${message.createdAt}
          )
          ORDER BY ${message.createdAt}
        ) FILTER (WHERE ${message.id} IS NOT NULL),
        '[]'::json
      )
    `,
  })
  .from(chat)
  .leftJoin(message, eq(message.chatId, chat.id))
  .where(eq(chat.userId, userId))
  .groupBy(chat.id, chat.title, chat.createdAt, chat.visibility)
  .orderBy(desc(chat.createdAt));
```

**Key points:**
- `leftJoin` instead of separate query (preserves chats with no messages)
- `json_agg` with `FILTER` to aggregate messages into array
- `COALESCE(..., '[]'::json)` returns empty array for chats with no messages
- `ORDER BY` inside json_agg maintains message order
- `groupBy` includes all selected chat columns
- Single query instead of 1 + N

**Alternative (if above SQL syntax issues):**
Use Drizzle's relational queries if schema has relations defined:
```typescript
const chatsWithMessages = await db.query.chat.findMany({
  where: eq(chat.userId, userId),
  with: {
    messages: {
      orderBy: message.createdAt,
    },
  },
  orderBy: desc(chat.createdAt),
});
```
(Only if schema.ts defines `relations(chat, ({ many }) => ({ messages: many(message) }))`)
