Create a new migration file that enables RLS and creates policies for 6 agent-related tables:

**Tables to protect:**
1. **ZyprusAgent** - Agents can read own record, admins can read all (Phase 19 will add admin checks)
2. **AgentChatSession** - Agents can only access their own sessions (agentId matches)
3. **PropertyListing** - Agents can access listings they submitted (submittedByAgentId matches) or are assigned to review
4. **LandListing** - Same as PropertyListing (agent-scoped access)
5. **ListingUploadAttempt** - Agents can only see attempts for their listings
6. **WhatsAppConversation** - Agents can only access conversations for their userId

**Policy patterns:**

**ZyprusAgent table:**
```sql
ALTER TABLE "ZyprusAgent" ENABLE ROW LEVEL SECURITY;

-- Agents can view own record (via userId mapping)
CREATE POLICY "Agents can view own record"
  ON "ZyprusAgent" FOR SELECT
  USING (userId = auth.uid());

-- Agents can update own record
CREATE POLICY "Agents can update own record"
  ON "ZyprusAgent" FOR UPDATE
  USING (userId = auth.uid())
  WITH CHECK (userId = auth.uid());

-- Note: INSERT/DELETE reserved for admin (Phase 19 will add admin policies)
```

**AgentChatSession:**
```sql
-- Agents can access their own sessions via ZyprusAgent.userId mapping
CREATE POLICY "Agents can view own sessions"
  ON "AgentChatSession" FOR SELECT
  USING (
    agentId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
  );

-- Similar for INSERT/UPDATE/DELETE
```

**PropertyListing and LandListing:**
```sql
-- Agents can view listings they submitted or are reviewing
CREATE POLICY "Agents can view own listings"
  ON "PropertyListing" FOR SELECT
  USING (
    submittedByAgentId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR firstReviewerId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR secondReviewerId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR userId = auth.uid()  -- Web app users can see their own listings
  );

-- Agents can insert listings (submittedByAgentId must match their ZyprusAgent.id)
CREATE POLICY "Agents can create own listings"
  ON "PropertyListing" FOR INSERT
  WITH CHECK (
    submittedByAgentId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR userId = auth.uid()  -- Web app users can create listings
  );

-- Agents can update listings they submitted or are reviewing
CREATE POLICY "Agents can update own listings"
  ON "PropertyListing" FOR UPDATE
  USING (
    submittedByAgentId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR firstReviewerId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR secondReviewerId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR userId = auth.uid()
  )
  WITH CHECK (
    submittedByAgentId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR firstReviewerId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR secondReviewerId IN (
      SELECT id FROM "ZyprusAgent" WHERE userId = auth.uid()
    )
    OR userId = auth.uid()
  );
```
