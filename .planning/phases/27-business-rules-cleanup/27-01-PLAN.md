### Step 4: Add "assign this to" pattern to email extraction (FR-11)

**File:** `supabase/functions/sophia-bot/handlers/email-webhook.ts`

**Problem:** `extractAssignmentFromEmail` regex matches "assign to" and "assign it to" but NOT "assign this to" or "assign listing to".

**Changes:**
Update the regex patterns to also match "this" and "listing":
```typescript
// Pattern 1: email
const emailMatch = lower.match(/assign(?:\s+(?:it|this|listing))?\s+to\s+(\S+@\S+)/);
// Pattern 2: offices
lower.includes(`assign to ${office}`) || lower.includes(`assign it to ${office}`) || lower.includes(`assign this to ${office}`)
// Pattern 3: names
const nameMatch = lower.match(/assign(?:\s+(?:it|this|listing))?\s+to\s+([a-z]+(?:\s+[a-z]+)?)/);
```

**Acceptance criteria:**
- [ ] "assign this to Susan" extracts susan@zyprus.com
- [ ] "assign listing to Diana" extracts diana@zyprus.com
- [ ] "assign to" and "assign it to" still work
