Search codebase for all environment variable references to ensure complete documentation:

```bash
# Find all Deno.env.get calls (Edge Functions)
# Use both quote and bracket notation to catch all patterns
grep -rE "Deno\.env\.(get\(|VARIABLE_NAME)" supabase/functions/ --include="*.ts" | \
  grep -oE 'Deno\.env\.get\(["\x27]([A-Z_]+)["\x27]\)|Deno\.env\[["'\'']([A-Z_]+)["'\'']\]' | \
  sed -E 's/.*["\x27]([A-Z_]+)["\x27].*/\1/' | sort -u > /tmp/edge-env-vars.txt

# Find all process.env references (Next.js)
# Catch both dot notation and bracket notation
grep -rE "process\.env\." {app,lib,components}/ --include="*.ts" --include="*.tsx" 2>/dev/null | \
  grep -oE 'process\.env\.([A-Z_]+)|process\.env\[["'\''"]([A-Z_]+)["'\''"]' | \
  sed -E 's/process\.env\.([A-Z_]+).*/\1/; s/.*["\x27]([A-Z_]+)["\x27].*/\1/' | sort -u > /tmp/nextjs-env-vars.txt

# Combine and deduplicate
cat /tmp/edge-env-vars.txt /tmp/nextjs-env-vars.txt | sort -u > /tmp/all-env-vars.txt
cat /tmp/all-env-vars.txt
```

**Structure:**
```bash
# ============================================
# SOPHIA AI Assistant - Environment Variables
# ============================================
#
# This file documents all required environment variables for local development.
# Copy this file to .env.local and fill in real values (never commit .env.local).
#
# For production Edge Functions, set secrets via:
#   supabase secrets set KEY=value --project-ref vceeheaxcrhmpqueudqx
#

# --------------------------------------------
# AI / LLM
# --------------------------------------------
# OpenRouter API key for AI model access (Gemini 3 Flash via proxy)
# Get from: https://openrouter.ai/keys
OPENROUTER_API_KEY=your_openrouter_api_key_here

# --------------------------------------------
# Database
# --------------------------------------------
# Supabase project URL
SUPABASE_URL=https://vceeheaxcrhmpqueudqx.supabase.co

# Supabase anonymous key (public, safe to expose in client-side code)
SUPABASE_ANON_KEY=your_supabase_anon_key_here

# Supabase service role key (NEVER expose client-side, server-only)
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key_here

# PostgreSQL connection URL (Session Pooler format for Next.js)
# Format: postgresql://postgres.[ref]:[password]@aws-0-eu-central-1.pooler.supabase.com:6543/postgres
POSTGRES_URL=your_postgres_connection_url_here

# --------------------------------------------
# Authentication (Next.js)
# --------------------------------------------
# NextAuth secret for JWT signing (generate with: openssl rand -base64 32)
AUTH_SECRET=your_auth_secret_here

# Base URL for auth callbacks (http://localhost:3000 for local dev)
AUTH_URL=http://localhost:3000

# --------------------------------------------
# WhatsApp Integration
# --------------------------------------------
# WaSenderAPI key for WhatsApp messaging (~$6/month)
# Get from: https://app.wasender.com
WASENDER_API_KEY=your_wasender_api_key_here

# Webhook secret for HMAC signature verification
# Set the same value in WaSenderAPI webhook configuration
WASENDER_WEBHOOK_SECRET=your_webhook_secret_here

# --------------------------------------------
# Telegram Integration
# --------------------------------------------
# Telegram Bot API token (from @BotFather)
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here

# Telegram group chat IDs for lead routing (v1.2 feature)
TELEGRAM_ZYPRESS_OTHERS_CHAT_ID=-1001234567890
VASYA_TELEGRAM_USER_ID=123456789

# Enable/disable Telegram bot (true/false)
SOPHIA_TELEGRAM_ENABLED=false

# --------------------------------------------
# Zyprus Property API
# --------------------------------------------
# Zyprus API OAuth client credentials
# Contact Zyprus admin for credentials
ZYPRUS_CLIENT_ID=your_zyprus_client_id_here
ZYPRUS_CLIENT_SECRET=your_zyprus_client_secret_here

# Zyprus API base URL (dev or production)
ZYPRUS_API_URL=https://dev9.zyprus.com

# --------------------------------------------
# Email (Resend)
# --------------------------------------------
# Resend API key for transactional email (document delivery)
# Get from: https://resend.com/api-keys
RESEND_API_KEY=your_resend_api_key_here

# --------------------------------------------
# 3CX Phone System (v1.2 feature)
# --------------------------------------------
# 3CX server URL and credentials for call log audit
CX3_BASE_URL=https://your-3cx-server.com
CX3_USERNAME=admin_username
CX3_PASSWORD=admin_password

# --------------------------------------------
# Observability
# --------------------------------------------
# Sentry DSN for error tracking (same for Next.js and Edge Functions)
# Get from: https://sentry.io/settings/projects/
SENTRY_DSN=https://your_sentry_dsn_here

# --------------------------------------------
# Cache & Rate Limiting
# --------------------------------------------
# Upstash Redis credentials for caching and rate limiting
# Get from: https://console.upstash.com/
KV_REST_API_URL=https://your-redis-instance.upstash.io
KV_REST_API_TOKEN=your_upstash_token_here

# --------------------------------------------
# Admin Access
# --------------------------------------------
# Secret for admin panel access (generate with: openssl rand -base64 32)
SOPHIA_ADMIN_SECRET=your_admin_secret_here

# --------------------------------------------
# Platform / Runtime
# --------------------------------------------
# Node environment (development/production)
NODE_ENV=development

# Deno deployment ID (auto-set by Supabase in production, leave empty for local)
DENO_DEPLOYMENT_ID=

# ============================================
# OPTIONAL ENVIRONMENT VARIABLES
# ============================================

# Vercel Blob storage (FUTURE - not currently deployed)
BLOB_READ_WRITE_TOKEN=your_vercel_blob_token_here
```

- Links to provider dashboards where applicable

- Local development defaults where applicable (AUTH_URL, NODE_ENV)
