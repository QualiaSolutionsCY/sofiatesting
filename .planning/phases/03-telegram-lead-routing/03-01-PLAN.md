1. Takes an optional region string (e.g., "Nicosia", "Famagusta", "Larnaca", "Paphos")
2. Returns the regional manager name from `REGIONAL_MANAGERS` map if region exists
3. Returns `null` if region is null/undefined or not found in the map (allows fallback)

3. Use regex pattern: `/\b(nicosia|famagusta|larnaca|larnaka|paphos|pafos|limassol|lefkosia|ammochostos)\b/i`
4. Map variants to canonical names (e.g., "larnaka" -> "Larnaca", "lefkosia" -> "Nicosia")

Export both functions.
