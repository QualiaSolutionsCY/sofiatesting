## What Was Done

- **Rebuilt the new-invoice composer.** Client field became a typeahead combobox
  (prefix search, auto-saves a new client on create, defaults to "add new",
  dropdown closed by default); removed Period, Qty, Property/reference, Billing
  address and Note fields; commission flag now defaults to "track agent". Why:
  client wanted a faster, leaner draft flow.
- **Added a 3-mode VAT control** (+ VAT / Including VAT / Excluding VAT) that now
  actually recalculates the live invoice and totals — the preview previously
  ignored the mode. Due date entered as days-to-pay, computed onto the invoice.
- **Redesigned the invoice template** to the client's exact letterhead: CSC Zyprus
  header + HE/CREA lines, Qty·Item·Description·Unit Price·Total ruled table that
  fills the A4 page, and a Balance Due row. Built an editable template (company
  details, Hellenic Bank banking block, settlement note) via a new
  `TemplateEditor` modal + `template-context` (localStorage-persisted).
- **Built the credit-note flow.** Same template, own CN number sequence, forced
  "Credit note for invoice №" description, no bank details; issuing one cancels the
  original invoice, prompts the operator for a reason, and sends the credit note to
  the group with that reason. Why: client's cancellation/audit requirement.
- **Admin auto-approval.** Invoices created in the admin panel are auto-approved
  and issued; Marios receives an "issued" WhatsApp notification instead of a
  review request (review gating stays only in the Sophia-bot chat flow).
- **Polished the left ledger list.** Rows show client · № · date · paid/unpaid ·
  amount; fixed selected-row black-on-black text; search now matches by amount;
  removed the stage chip; widened the column.
- **Misc:** draft numbers now follow the real sequence with a `-DRAFT` suffix;
  "Numbered"→"Approved" and "Paid·sent"→"Receipt" stage renames; `.00` trimmed
  from amounts; print/download scoped so the PDF prints only the invoice; monthly
  run got a real PDF batch preview + working "Create"; command palette reduced to
  options/pages + a yearly-run entry; all test invoices emptied from the store.
