## v1 Requirements

Requirements for 3CX Call Log Audit integration.

### 3CX Integration

- [x] **3CX-01**: System can login to 3CX web interface with provided credentials
- [x] **3CX-02**: System can navigate to call log section and extract today's calls
- [x] **3CX-03**: System can filter calls to target number (22032770) only
- [x] **3CX-04**: System can exclude internal extensions (70, 64, 99, 801, 900)
- [x] **3CX-05**: System can extract caller phone numbers from call log table
- [x] **3CX-06**: System can handle web session management and authentication errors

### Telegram Integration

- [x] **TG-01**: System can search phone numbers across 4 regional Telegram groups
- [x] **TG-02**: System can send formatted alerts to "Zypress Others" group
- [x] **TG-03**: System can track message responses from Vasya
- [x] **TG-04**: System can handle alternative phone number scenarios
- [x] **TG-05**: System can stop follow-up when Vasya responds positively

### Call Tracking

- [x] **TRACK-01**: System can store processed calls in database with timestamps
- [x] **TRACK-02**: System can track alert status for each missing caller
- [x] **TRACK-03**: System can prevent duplicate processing of same day's calls
- [x] **TRACK-04**: System can implement 24-hour follow-up logic
- [x] **TRACK-05**: System can maintain conversation state per phone number

### Scheduling

- [x] **SCHED-01**: System can execute audit daily Monday-Friday at 5:00 PM Cyprus time
- [x] **SCHED-02**: System can skip execution on weekends
- [x] **SCHED-03**: System can log execution status and errors
- [x] **SCHED-04**: System can handle timezone calculations correctly
- [x] **SCHED-05**: System can recover from failed executions

### Alerting

- [x] **ALERT-01**: System can send initial missing caller alerts using specification template
- [x] **ALERT-02**: System can send follow-up reminders after 24 hours
- [x] **ALERT-03**: System can format messages with call time and phone number
- [x] **ALERT-04**: System can handle response scenarios per specification
- [x] **ALERT-05**: System can stop alerting when number is found in groups

## v2 Requirements (Deferred)

### Analytics
- ANLY-01: Generate daily/weekly call audit reports
- ANLY-02: Track missed call patterns and response rates
- ANLY-03: Monitor system performance and success rates

### Manual Controls
- MNCT-01: Allow manual triggering of audits outside schedule
- MNCT-02: Provide admin interface for managing tracked calls
- MNCT-03: Support manual override of alert states

## Out of Scope

| Feature | Reason |
|---------|--------|
| Real-time call monitoring | Daily audit sufficient per specification |
| Call recording integration | Not required for lead tracking |
| Multi-location 3CX systems | Single system specified |
| Complex reporting dashboard | Simple alerting sufficient |
| SMS/WhatsApp alerts | Telegram specified in requirements |
