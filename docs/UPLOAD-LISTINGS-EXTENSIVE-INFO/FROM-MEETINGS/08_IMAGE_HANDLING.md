# Image Handling

## Overview

Photos are critical for property listings. Sophia must handle images carefully, ensuring proper ordering, privacy protection, and quality presentation.

## Image Sources

Agents may send images via:
- WhatsApp attachments
- Email attachments
- Links to Bazaraki listings
- Links to other websites
- Google Drive/cloud storage links

## CRITICAL: Privacy Checks

Before uploading ANY photo, check for and handle these privacy concerns:

### Must NOT be Visible in Photos:
- ❌ Exact street address
- ❌ Building number/name that identifies exact location
- ❌ Door numbers
- ❌ Car registration plates
- ❌ Personal photos of residents/owners
- ❌ Documents with personal information
- ❌ Names on mailboxes/doorbells

### Action When Privacy Issues Found:
1. **Crop the image** to remove identifying information
2. If cropping not possible, **blur the area** or **exclude the image**
3. Note to reviewer if image quality is compromised

## Bazaraki Logo Removal

When images come from Bazaraki:
- They often have **Bazaraki watermark/logo**
- **Crop 2-3cm from all sides** to remove logos
- Ensure main property content remains visible after cropping

## Image Ordering

Photos must be uploaded in a **specific order** for best presentation.

### Order Priority (First to Last):

1. **Best External Photo** - Front of property, most attractive exterior shot
2. **Other External Photos** - Garden, additional exterior angles, street view
3. **Pool/Garden Features** - Swimming pool, landscaped areas, terraces
4. **Most Spacious Interior** - Usually living room, open plan area
5. **Kitchen** - Full kitchen shot
6. **Additional Living Spaces** - Dining room, second living area
7. **Bedrooms** - Master bedroom first, then others
8. **Bathrooms** - En-suite first, then family bathroom
9. **Other Rooms** - Office, storage, utility
10. **Building Amenities** - Lobby, elevator, communal areas (last priority)

### First Photo Rule
The **first photo** is crucial - it's the thumbnail shown in search results. Always choose:
- Most attractive exterior shot
- Good lighting
- Shows property's best feature
- High quality image

## Image Quality Guidelines

### Ideal Images:
- ✅ Well-lit (natural daylight preferred)
- ✅ High resolution
- ✅ Properly oriented (not sideways/upside down)
- ✅ Shows full room/space
- ✅ Clean, tidy spaces
- ✅ No extreme wide-angle distortion

### Acceptable but Not Ideal:
- ⚠️ Slightly dark (can still see details)
- ⚠️ Medium resolution
- ⚠️ Minor clutter visible

### Avoid/Exclude:
- ❌ Blurry images
- ❌ Very dark photos where details invisible
- ❌ Sideways or upside down (rotate if possible)
- ❌ Duplicate/near-duplicate images
- ❌ Screenshots of photos (quality loss)
- ❌ Photos that don't show property features

## Uploading Process

### Method: Drag and Drop
1. Access the gallery/images section of the listing form
2. Drag images in the correct order
3. Drop to upload
4. Reorder by dragging if needed

### File Formats Accepted:
- JPEG/JPG
- PNG
- WebP (may need conversion)

### File Size:
- If files too large, compression may be needed
- Aim for good quality at reasonable size

## Image Scanning for Features

If Sophia has image recognition capabilities, scan photos to identify:

### Features to Look For:
- **AC Units** - Wall-mounted air conditioning
- **Shutters** - Electric or manual
- **Pool** - Private or communal
- **Garden** - Landscaped, lawn, potted plants
- **Parking** - Covered, carport, garage
- **Solar Panels** - On roof
- **Satellite Dish** - TV/internet
- **BBQ Area** - Built-in or portable
- **Sea/Mountain Views** - Visible from property
- **Modern Kitchen** - Style, appliances
- **Quality Finishes** - Marble, granite, wood floors

### How to Use Scanned Information:
- Add identified features to the description
- Tick relevant feature checkboxes
- Only include if confident in identification

**Example:**
- See AC units on walls in bedroom photos → tick "Air Conditioning" feature
- See landscaped garden in exterior shot → mention in description
- See pool in background → verify if private or communal before adding

## Minimum Images Required

- **Minimum:** 1 photo (not ideal)
- **Recommended:** 8-15 photos
- **Maximum:** System limit varies

If agent sends only 1-2 photos, Sophia may ask:
> "Do you have any additional photos of the property? More images help attract potential buyers."

## Handling Missing Images

If no images provided:
1. Request images from agent before uploading
2. Cannot upload listing without at least 1 image
3. Message: "Could you please send me some photos of the property? I need at least one image to create the listing."

## Example Image Set for 2-Bed Apartment

Ideal order:
1. External - Building front with pool visible
2. External - Communal pool area
3. External - Covered balcony with view
4. Interior - Open plan living/dining
5. Interior - Kitchen
6. Interior - Living area alternate angle
7. Interior - Master bedroom
8. Interior - Second bedroom
9. Interior - Master en-suite bathroom
10. Interior - Family bathroom
11. Building - Lobby/entrance
12. Building - Elevator (if applicable)

## What Agents Usually Send vs. What's Needed

### Agents Often Send:
- Mixed order of random photos
- Some blurry images
- Bazaraki screenshots
- Too many similar photos

### What Sophia Should Do:
1. Sort into correct order
2. Remove/exclude poor quality images
3. Crop Bazaraki logos
4. Use best image as first photo
5. Ask for more photos if insufficient
